#### [无重复字符的最长子串](https://leetcode-cn.com/explore/learn/card/hash-table/207/conclusion/826/)

> 给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。
示例 1:
```bash
输入: "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
```
